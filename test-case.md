 ### Тест-кейс 1. Создание задачи

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Кликнуть на кнопку **«Создать задачу»** | Открыто окно для ввода данных для новой задачи |
| 2 | Ввести название задачи | Поле доступно для ввода, название вписано и отображается |
| 3 | Ввести описание задачи | Поле доступно для ввода, описание вписано и отображается |
| 4 | Выбрать проект | Проект указан |
| 5 | Выбрать приоритет | Приоритет выбран |
| 6 | Выбрать исполнителя | Исполнитель выбран |
| 7 | Нажать кнопку **«Создать»** | На странице с задачами успешно отображается новая задача с введёнными данными |
                                                                                                              


### Тест-кейс 2. Открытие карточки задачи

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть страницу со списком задач | Страница со списком задач успешно загружена |
| 2 | Найти существующую задачу в списке | Задача отображается в списке |
| 3 | Кликнуть по названию задачи | Открывается карточка выбранной задачи |
| 4 | Проверить отображение названия задачи | Название задачи отображается корректно |
| 5 | Проверить отображение описания задачи | Описание задачи отображается корректно |
| 6 | Проверить отображение проекта | Проект указан корректно |
| 7 | Проверить отображение приоритета | Приоритет указан корректно |
| 8 | Проверить отображение статуса задачи | Статус задачи отображается корректно | 
| 9 | Проверить отображение исполнителя | Исполнитель указан корректно |

### Тест-кейс 3. Поиск задачи

| № | Шаг  | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть страницу со списком задач | Страница со списком задач успешно загружена
| 2 | Ввести ключевое слово в поле поиска | Отображены результаты поиска с ключевым словом |

### Тест-кейс 4. Переход на доску проекта

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть страницу **«Проекты»** | Страница «Проекты» успешно загружена |
| 2 | Найти нужный проект в списке проектов | Проект отображается в списке |
| 3 | Нажать кнопку **«Перейти к доске»** у выбранного проекта | Происходит переход на страницу доски проекта |
| 4 | Проверить, что открылась доска выбранного проекта | Отображается доска именно выбранного проекта |
| 5 | Проверить отображение колонок на доске проекта | Колонки доски отображаются корректно |
| 6 | Проверить отображение задач в колонках | Задачи проекта отображаются в соответствующих колонках |
